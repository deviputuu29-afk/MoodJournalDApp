# Mood Journal DApp

**Mood Journal DApp** – Blockchain-Based Emotional Journal and Daily Reflection System

---

## Project Description

Mood Journal DApp is a decentralized journaling application built on the Stellar blockchain using the Soroban SDK and Rust programming language. The system enables users to securely store, manage, and organize their daily emotional reflections directly on the blockchain without relying on centralized platforms.

This decentralized application allows users to create personal journal entries containing mood categories, dates, and daily notes. By leveraging blockchain technology, the application ensures transparency, data persistence, and user ownership while maintaining a lightweight and efficient architecture.

The smart contract stores all journal data within Soroban instance storage, enabling users to interact with their emotional records securely and permanently through blockchain-based operations.

  Eliminate dependence on centralized databases by using decentralized smart contracts.

* **Support Mental Wellness Through Technology**

# Smart Contract Functions

## `create_journal()`

Creates a new journal entry containing:

* date
* mood
* note content

---

## `get_journals()`

Returns all stored journal entries from blockchain storage.

---

## `delete_journal()`

Deletes a journal entry using its unique identifier.
